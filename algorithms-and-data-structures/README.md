# Algorithms and Data Structures

Implementation of core data structures in C++ developed as part of university coursework. The projects focus on low-level memory management, generic programming with templates, and performance analysis of fundamental algorithms and data structures.

## Implemented Structures

- Linked List
- Dynamic Array
- Binary Search Tree (BST)
- Binary Heap (Priority Queue)
- Hash Table (Separate Chaining)

## Key Concepts

- Generic programming (C++ templates)
- Dynamic memory allocation
- Tree and heap operations
- Hashing and collision handling
- Time complexity and performance benchmarking

## Technologies

- C++
- Object-Oriented Programming (OOP)
