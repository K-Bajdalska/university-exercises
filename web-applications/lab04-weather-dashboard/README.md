# Weather Dashboard

University assignment created for the Web Applications course.

## Description

A weather application built with JavaScript that retrieves current weather conditions and forecast data from the OpenWeatherMap API.

The application allows users to search for a city and displays:

- current temperature,
- perceived temperature,
- wind speed,
- visibility,
- weather description,
- forecast entries returned by the API.

The interface automatically switches between day and night themes based on the weather icon returned by the service.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- XMLHttpRequest
- OpenWeatherMap REST API

## Features

- city search
- current weather display
- weather forecast display
- dynamic DOM generation
- day/night theme switching
- JSON data processing
