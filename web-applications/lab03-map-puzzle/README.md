# Lab 03 - Map Puzzle

Interactive map-based puzzle game created as part of the Web Applications course.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js

## Features

- Interactive map using Leaflet
- User geolocation support
- Map image generation
- Dynamic puzzle creation from the selected map area
- Drag and drop puzzle solving
- Browser notifications
- Canvas-based image processing

## Learning Goals

This project focuses on:

- Working with third-party JavaScript libraries
- Geolocation API
- Notifications API
- Drag and Drop API
- Canvas manipulation
- Event-driven programming
- Dynamic DOM generation
