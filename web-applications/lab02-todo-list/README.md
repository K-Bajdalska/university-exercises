# Lab 02 - To-Do List

Interactive To-Do List application created as part of the Web Applications course.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search and filter tasks
- Data persistence using Local Storage
- Basic input validation

## Learning Goals

This project focuses on:

- DOM manipulation
- Event handling
- Object-oriented JavaScript
- Local Storage API
- Form validation
