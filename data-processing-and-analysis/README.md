# Data Processing and Analysis

University laboratory project focused on core data science techniques, including signal processing, information theory, and dimensionality reduction.

## Key Skills

- Signal processing: sampling, aliasing, quantization
- Image processing: grayscale conversion, histogram analysis, Otsu thresholding
- Information theory: entropy, mutual information, information gain
- Feature selection using information gain (Zoo dataset analysis)
- Dimensionality reduction: PCA implemented from scratch and using scikit-learn
- Data visualization and exploratory analysis

## Implemented Methods

- Custom implementation of probability and entropy functions
- Information gain calculation for feature evaluation
- Principal Component Analysis (PCA) from scratch using eigen decomposition
- Comparison with `sklearn.decomposition.PCA`
- Reconstruction error analysis for dimensionality reduction

## Tools & Libraries

- Python (NumPy, Pandas)
- Matplotlib
- OpenCV
- scikit-learn
- PIL

## Summary

Project demonstrates practical understanding of data processing fundamentals and core machine learning preprocessing techniques, implemented both manually and using standard scientific libraries.
