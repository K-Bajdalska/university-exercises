Console-based management system for a restaurant written in C as a first-semester programming project.

The application uses a modular structure (multiple .c and .h files) and simulates real-world restaurant management processes such as employee handling, warehouse management, supplier control, and role-based access to system functionalities.

Features:
User login system with role-based permissions (4 user types)
Employee management (add/remove, statistics, evaluations)
Warehouse module with inventory simulation and stock control
Supplier management system
Restaurant information panel with editable data (admin access)
File-based data persistence (.txt files)
Dynamic memory allocation and manual memory management
